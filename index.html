Tecnologías

HTML

HTML es el lenguaje utilizado para crear la estructura de las páginas web mediante etiquetas, textos, imágenes y enlaces.

CSS

CSS permite diseñar y dar estilos visuales a una página web, mejorando colores, tamaños, posiciones y animaciones.

JavaScript

JavaScript agrega interactividad a las páginas web mediante eventos, validaciones, efectos dinámicos y funciones.

PHP

PHP es un lenguaje de programación para el desarrollo backend, utilizado para crear sistemas web dinámicos y conectados a bases de datos.
